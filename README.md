# Backend-SErver

Este será nuestro backend api rest conectado a MongoDB usando Mongoose.

Para ejecutarla, es necesario reconstruir los módulos de node usando el comando

```
npm install
```

## Dentro de Google-Signin-demo
Existe un pequeño app para probar la autenticación de Google en un Front-End básico pero funcional. Deberemos de hacer lo mismo que más arriba para instalar los módulos de node.
Además para arrancarlos deberemos de escirbir 

```
npm run dev
```
